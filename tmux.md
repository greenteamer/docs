# Tmux guide

### Tmux Sessions

`tmux new -s <session name>` - create new session  
`tmux attach -t <session name>` - jump to session  
`Ctrl-a-d` - move out from session  
`Ctrl-a-x` - close window  


### Tmux Windows

`Ctrl-a-v` - split window vertical  
`Ctrl-a-V` - split window horizontal  
*move to window*
`Ctrl-a-h`  
`Ctrl-a-j`  
`Ctrl-a-k`  
`Ctrl-a-l`
